# Deploy to Vercel 🚀

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/DixenM/Oceanmapping)

## Quick Deploy from iPhone

1. Click the "Deploy with Vercel" button above
2. Sign in with GitHub
3. Add environment variable:
   - `VITE_WORLDTIDES_API_KEY` = your API key
4. Click Deploy
5. Done! 🎉
